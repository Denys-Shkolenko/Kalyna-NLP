# Kalyna-NLP
Text-to-speech Telegram Bot
  
### Created using:
- java
- spring boot
- telegram bot library for java (https://github.com/rubenlagus/TelegramBots)
- google cloud text-to-speech api
- google cloud speech-to-text api
